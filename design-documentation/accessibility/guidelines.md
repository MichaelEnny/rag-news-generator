# Accessibility Guidelines

## Core Requirements
- Keyboard navigation must reach all controls.
- Focus indicators must remain visible against every surface.
- Status communication must not rely on color alone.
- Form validation text must be tied to fields.
- Motion should respect `prefers-reduced-motion`.
