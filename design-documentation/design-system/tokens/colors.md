# Color Tokens

## Tokens
- `--canvas`: `#f7f1e8`
- `--canvas-strong`: `#efe3d3`
- `--ink-950`: `#0f1720`
- `--ink-900`: `#17222e`
- `--ink-800`: `#243241`
- `--ink-700`: `#34506a`
- `--rust-500`: `#b74f2b`
- `--rust-600`: `#8e3314`
- `--gold-400`: `#e7b53f`
- `--teal-500`: `#4e8f87`
- `--success-500`: `#2d7d53`
- `--warning-500`: `#b7791f`
- `--error-500`: `#a33a2f`

## Usage Notes
- Ink colors establish information hierarchy.
- Rust and gold are reserved for emphasis and operator attention.
