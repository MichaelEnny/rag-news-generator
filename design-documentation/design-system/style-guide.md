# Design System Style Guide

## Overview
The interface uses an editorial palette built from paper tones, dark ink surfaces, and warm accent colors. The goal is to make operational data feel legible and intentional, not sterile.

## Color
- Canvas: `#f7f1e8`
- Canvas strong: `#efe3d3`
- Ink: `#0f1720`, `#17222e`, `#243241`
- Rust accent: `#b74f2b`
- Gold accent: `#e7b53f`
- Teal accent: `#4e8f87`

## Typography
- Display: `Space Grotesk`
- Body: `Source Sans 3`
- Mono: `IBM Plex Mono`

## Layout
- Three-column metric grid on wide viewports.
- Two-region dashboard split: operational rail plus digest canvas.
- Single-column collapse on smaller screens.

## Interaction
- Primary actions are dark ink buttons.
- Secondary actions use rust accent treatment.
- Focus states use a gold outline with strong contrast.
